# Typst template for DHBW or other universities (based on the LaTeX template)

This template is a Typst implementation modeled after the layout/look of the common DHBW LaTeX template (title page, lists, numbering, typical structure). It can, however, also be adapted for other universities.

## Why Typst and not LaTeX?

- **Faster workflow:** short compile times, quick feedback loops
- **Readable syntax:** less boilerplate, more focus on content
- **Better error messages:** issues are usually easier to track down
- **Modern layouting:** tables/figures/grids are consistently integrated
- **Less package sprawl:** a lot works without "wait, which package was that again?"

## Usage

Until this is available on Typst Universe, it can be added locally to these folders:

- Linux: ~/.local/share/typst/packages/local/turbocharged-dhbw/0.2.0
- Windows: Once you're done with the 4-hour forced update, it's here: %APPDATA%\typst\packages\local\turbocharged-dhbw\0.2.0
- macOS: You'll figure it out. (Or just buy a decent computer.)
